## Usage

**Help**:<br>
```python
fastmail.py --help
```
<br>
**Run normally**:
```python
fastmail.py
```
<br>
**Load tempmail from saved file:**
```python
fastmail.py -f <email data file>
```
<br>
**Delete tempmail from saved file**:
```python
fastmail.py -d <email data file>
```
